# Currículum Vitae Interactivo - Evaluación Final Módulo 2

Este es el proyecto final para el módulo nùmero 2. Consiste en un CV profesional  responsivo, interactivo y con validaciones en tiempo real.

##  Demo
Puedes ver el proyecto funcionando aquí: [PEGA AQUÍ TU ENLACE DE GITHUB PAGES]

## Tecnologías Utilizadas
* **HTML5:** Estructura semántica.
* **CSS3:** Estilos personalizados y Google Fonts.
* **Bootstrap 5:** Sistema de rejilla y diseño responsivo.
* **JavaScript & jQuery:** Interactividad, animaciones y validación.

##  Funcionalidades Implementadas
1. **Navegación Fluida (Smooth Scroll):** Desplazamiento suave entre secciones mediante jQuery.
2. **Modo Oscuro/Claro:** Botón que alterna el tema visual de toda la página.
3. **Validación de Formulario:** Comprobación en tiempo real de los campos de contacto.
4. **Diseño Responsivo:** Adaptado para dispositivos móviles y escritorio.
